Blog创建指南

2025年08月22日

大家好，顾名思义，这是一篇教您如何建立blog的文章.俗语道:

前人栽树，后人乘凉

在我5个小时的试错后终于建立了我的blog并且总结了这篇文章的内容，希望可以帮助到您.

本文是凭借记忆还原的，如果有错漏务必指正.

**Warning：

下文默认您使用的是Windows系统

本文主题将会使用nexmoe为例

如果出现问题请先进行搜索，99%的问题都可以搜到答案

有的问题是由于网络不佳，建议等会再试或使用VPN**

### Chapter 1 安装Node.js

在[这里](https://nodejs.org/en/download/ “这里”)下载即可，点击*Windows Installer*.使用这个Installer进行安装即可.

### Chapter 2 安装Git

在[这里](https://mirrors.tuna.tsinghua.edu.cn/github-release/git-for-windows/git/Git%20for%20Windows%20v2.51.0.windows.1/ “这里”)下载即可，点击*Git-2.51.0-64-bit.exe*.使用这个exe进行安装即可.

### Chapter 3 安装Hexo

接下来您应当以管理员身份运行软件*Windows PowerShell*，如若您没有该软件，也可以点击Win+R，输入cmd代替，但是还是建议您使用该软件.

然后输入**npm install -g hexo-cli**以下载Hexo

如果网络不好可以使用**VPN**

接着在**C盘以外**的地方建立一个文件夹，例如在D盘建立“blog”

在cmd/Shell内输入**cd D:\ **,再输入**cd blog**

(请根据自己的情况适当更改以上两步)

继续安装Hexo,输入**hexo init myblog**.您可将myblog换成想要的名字，最好使用英文.

### Chapter 4 安装nexmoe并配置

输入**npm install hexo-theme-nexmoe hexo-renderer-inferno**

接下来您可以在D:\blog\myblog(请根据自己的情况适当更改)中找到一个叫做**_config.yml**的文件

使用文本文档进行编辑，找到“theme” 在后面加上nexmoe

注意，冒号与“nexmoe”间有空格.

不要忘记保存.

### Chapter 5 测试

输入**hexo server –debug**

如果没有错误反馈并且能够成功进入[这里](http://localhost:4000/ “这里”)，那么您之前的步骤是正确的，如果不可以进入请搜索错误报告进行捉虫.

### Chapter 6 准备Github

进入[这里](https://github.com/ “这里”)，点击右上角sign up，跟随指导注册账号并进入邮箱验证.

接下来点击右上角加号，选择Create new repositories.

将仓库名称命名为**您的用户名.github.io**，下方选择Public.

### Chapter 7 发布blog

输入**git config –global user.name “您的用户名”**(注意引号不要落下)

输入**git config –global user.email “您的邮箱地址”**

接下来把**_config.yml**中最后的deploy改成这样的格式

别忘了缩进和保存.

deploy:

  type: ‘git’

  repo: ‘这里放您的HTTPS’

  branch: main

 

HTTPS可以在您的GitHub仓库中找到.

接下来先确保您的Shell/cmd正在您的blog所在文件夹中.

输入**npm install hexo-deployer-git –save**

然后依次输入**hexo clean**,**hexo g**,**hexo d**.

### Chapter 8 测试

如果没有错误反馈，GitHub仓库中出现文件，并且能够成功进入**您的用户名.github.io**这个网站，那么您之前的步骤是正确的，恭喜！

到这里，您的blog已经成功上传了，未来的步骤请您自己探索吧！
